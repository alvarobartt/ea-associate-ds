Un code-barres, ou code à barres, est la représentation d'une donnée numérique ou alphanumérique sous forme d'un symbole constitué de barres et d'espaces dont l'épaisseur varie en fonction de la symbologie utilisée et des données ainsi codées. Les codes-barres ont été créés en 1970 par George Laurer.Lorsque ces barres sont remplacées par de petits carrés ou points, on parle de code en deux dimensionsLes codes-barres sont destinés à une lecture automatisée par un capteur électronique, le lecteur de code-barres.SymbologiesQuelques types de codes-barres unidimensionnels (1D) :les codes-barres EAN : EAN 8 , EAN 13 , Code Universel des Produits (CUP)le Codabar Monarchle Code 11le Code 39le Code 93le Code 128le 2 parmi 5Quelques types de codes-barres bidimensionnels (2D) :le PDF-417le Code 16Kle Code 49le Code Onele DataMatrix surnommé Tag pour certaines applications.le Flashcode (spécification issue de DataMatrix)le MaxiCodele Code Aztecle Code QRNotesLiens externesGénérateur de Code à Barre (Gratuit)Générateur de code barre format vectoriel (pdf) sur la base d'un ISBN (gratuit)Générateur de code-barres (lien commercial)Barby : Librairie libre de génération de codes barres (en Ruby)Polices et générateurs de code-barre (sous license GPL)Blog francophone sur les codes graphiques, code 1D, 2D associés au téléphone portable*ar:شفرة خيطية az:Barkod bat-smg:Barkods bg:Баркод ca:Codi de barres cs:Čárový kód da:Stregkode de:Strichcode en:Barcode es:Código de barras et:Vöötkood fa:بارکد fi:Viivakoodi gl:Código de barras he:ברקוד hr:Crtični kôd hu:Vonalkód id:Kode batang it:Codice a barre ja:バーコード ko:바코드 lt:Brūkšninis kodas lv:Svītrkods ms:Kod bar nl:Streepjescode no:Strekkode pl:Kod kreskowy pt:Código de barras ro:Cod de bare ru:Штриховой код sk:Čiarový kód sl:Črtna koda sq:Barkodi sr:Бар код sv:Streckkod th:บาร์โค้ด tr:Barkod uk:Штрих-код vi:Mã vạch zh:条形码
