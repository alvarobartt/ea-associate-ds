 WordPad est un traitement de texte basique qui est inclus dans toutes les versions de Windows à partir de 95.Il permet une mise en forme basique et l'impression. Il ne comprend ni de correcteur orthographique, ni de gestion des styles. Il est bon pour écrire des lettres ou des textes courts mais n'est pas assez puissant pour de nombreuses tâches ou pour rédiger de longs rapports avec des graphiques ou de longs textes, tels des thèses ou des livres.Format de sauvegarde des donnéesWordPad est compatible avec le format médiocrement normalisé RTF, ainsi qu'avec les fichiers "Word 6.0 pour Windows" et les versions précédentes de Word. WordPad pour Windows XP ne peut sauvegarder sous le format DOC (mais seulement TXT ou RTF).WordPad est le seul produit officiellement maintenu par Microsoft (sauf Microsoft Word) qui peut lire ou convertir des fichiers WRI créés par Microsoft Write mais ne peut pas sauvegarder sous ce format.PositionnementWordPad a été introduit en premier lieu dans Windows 95 pour remplacer le vieux Microsoft Write qui était intégré dans les anciennes versions de Windows. Le code-source de WordPad est distribué par Microsoft comme exemple dans Microsoft foundation class 3.2 et plus, légèrement avant la sortie de Windows 95. Il est toujours disponible à télécharger sur le site MSDN http://msdn.microsoft.com/library/default.asp?url=/library/en-us/vcsample/html/_sample_mfc_WORDPAD.asp.WordPad bénéficie de sa pré installation en tant que composant du système Windows.Si on le compare à d'autres logiciels de traitement de texte pour windows, disponibles gratuitement, comme AbiWord ou OpenOffice.org Writer, il est nettement plus limité.Cette limitation pourrait être volontaire de la part de Microsoft, afin d'inciter à l'achat de Microsoft Word.Catégorie:Composant Windows Catégorie:Traitement de textecs:WordPad de:WordPad en:WordPad es:WordPad fi:Microsoft WordPad it:Microsoft WordPad ja:ワードパッド ko:워드패드 nl:Microsoft WordPad pl:WordPad pt:WordPad ro:WordPad ru:WordPad simple:WordPad sq:WordPad sv:Microsoft WordPad uk:WordPad zh:WordPad
